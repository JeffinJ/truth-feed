# Truths Feed 

A real-time feed of Donald Trump's posts on Truth Social

## Development setup:

```bash
# development
npm run dev

# build
npm run build

# start
npm run start

```


## Environment

`.env.local`

```bash
NEXT_PUBLIC_TRUTH_API=
NODE_TLS_REJECT_UNAUTHORIZED=0
```
